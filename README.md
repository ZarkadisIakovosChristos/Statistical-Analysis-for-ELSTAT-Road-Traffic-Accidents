# Statistical-Analysis-for-ELSTAT-Road-Traffic-Accidents
This repository contains information and material for the Statistical Analysis for ELSTAT Road Traffic Accidents, a project for the course "Actuarial Modeling" (UNIPI, Department of Statistics and Insurance Science). The purpose of the project is to collect data from the Hellenic Statistical Authority regarding road traffic accidents and perform statistical analysis, like descriptive statistics, t-tests and distribution fitting. The project was written in R
